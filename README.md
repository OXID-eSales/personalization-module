Econda module
=============

Module adds Econda service functionality.

## Requirements

* OXID eShop compilation ^v6.1.0

## Installation

* Register module repository in project root `composer.json` file.
* Execute command: `composer require oxid-esales/econda-module:^1.0.0`.
